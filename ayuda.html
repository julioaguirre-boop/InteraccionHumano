Ayuda

Como funciona

Esta aplicacion permite registrar, organizar y dar seguimiento a tareas escolares. Puedes agregar tareas con materia, prioridad y fecha, facilitando el control de actividades.

Objetivo

Mejorar la organizacion academica, reducir el estres y aumentar la productividad.

Guia de uso

1. Agrega una tarea.
2. Selecciona materia.
3. Elige prioridad.
4. Define fecha.
5. Guarda.

Puedes marcar tareas como completadas o eliminarlas.
